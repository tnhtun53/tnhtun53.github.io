# Portfolio Website

Welcome to my portfolio website! This is a simple, clean, and modern portfolio built with HTML, CSS, and JavaScript.

## 🌐 Live Site

Visit: [https://tnhtun53.github.io](https://tnhtun53.github.io)

## 📋 Features

- **Responsive Design** - Looks great on all devices (mobile, tablet, desktop)
- **Modern Styling** - Clean and professional UI with smooth animations
- **Projects Showcase** - Display your best work and projects
- **Skills Section** - Highlight your technical skills
- **Contact Links** - Easy way for visitors to get in touch
- **Dark Theme** - Eye-friendly dark color scheme

## 🚀 Getting Started

### Prerequisites
- Git
- A GitHub account

### Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/tnhtun53/tnhtun53.github.io.git
   cd tnhtun53.github.io
   ```

2. Edit the files to personalize your portfolio:
   - Update `index.html` with your information
   - Modify `styles.css` to change colors and styling
   - Add your projects and skills

3. Push your changes to GitHub:
   ```bash
   git add .
   git commit -m "Customize portfolio"
   git push origin main
   ```

4. Your site will be live at `https://tnhtun53.github.io` (may take a few minutes to update)

## 🎨 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0f172a;
    --secondary-color: #1e293b;
    --accent-color: #06b6d4;
    --text-color: #f1f5f9;
    --muted-text: #cbd5e1;
}
```

### Add Your Projects
Edit the Projects section in `index.html` and update:
- Project title
- Description
- Technologies used
- Link to project

### Update Contact Information
Replace the email and social media links in the Contact section with your own.
